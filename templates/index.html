Introduction

A brief introduction to the development site

This site has been created to store the data files and additional tools used to create the CPAN Testers family of websites.

Please note that although this site is dynamically generated, only the database file sizes are updated. The links below are generic, and will access the appropriate files as they are updated.

CPAN Testers Statistics Database Generator

Downloads for the CPAN Testers 'cpanstats' database

The following data files and tools are provided for use in building and expanding the data storage, used by the CPAN Testers family of websites, containing the reports metadata as posted to the cpan-testers mailing list by the testers. If you have suggestions for improving the code or databases, please use the RT queue. Uncompressed size of the SQLite database is [% CPANSTATSDB %].

  • CPAN-Testers-Data-Generator (GitHub)
  • Latest SQLite database: Gzip ([% CPANSTATSGZ %]) / Bzip2 ([% CPANSTATSBZ %])
  • ORDB-CPANTesters
